# 🔊 Sound Shop | Professional Music Licensing

**Sound Shop** is a high-end digital storefront for cinematic instrumental soundtracks. This project features a custom-built mobile-first web interface with integrated secure payment processing and digital licensing agreements.

## 🚀 Features
* **Cinematic Catalog:** High-quality instrumental tracks for media licensing.
* **Automated Licensing:** Dynamic on-page contract signing with electronic signature support.
* **Secure Checkout:** Integrated with **Authorize.net** via Node.js backend.
* **Mobile-First Development:** Entirely developed and managed within a **Termux** environment on Android.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Neon Glassmorphism Aesthetic), JavaScript (Accept.js).
* **Backend:** Node.js, Express.js.
* **Environment:** Termux, Git, GitHub.

## ⚖️ Licensing
Every purchase includes a legally binding license agreement, generated at the time of checkout.
